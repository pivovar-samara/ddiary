# Security Policy

## Supported versions

This project is maintained on a best-effort basis. Security fixes are applied
to the latest mainline code.

## Reporting a vulnerability

Please report vulnerabilities privately by email to:

- `plantry.sup@gmail.com`

Do not open public GitHub issues for security-sensitive reports.

When possible, include:

- A clear description of the issue and impact.
- Reproduction steps or a proof of concept.
- Any suggested remediation.

You will receive an acknowledgment as soon as possible, and we will coordinate
validation, remediation, and disclosure timing.
